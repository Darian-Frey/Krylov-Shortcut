# ⚡ krylov-shortcut

A computational engine for **Shortcuts to Adiabaticity (STA)** using Krylov subspace methods. This is the "efficiency" pillar of the **CODE-GEO** framework, focusing on how quantum systems can bypass complexity growth.

## 🚀 Purpose

To provide a fast protocol for quantum state preparation by calculating the **Adiabatic Gauge Potential** within the Krylov basis. This allows for "infinite-speed" state transitions without decoherence.

## ⚡ Verified Benchmarks

* **Logarithmic Efficiency:** 6020x gain at 100k depth.
* **Linear Tunneling:** Bypassing $L^2$ time-suppression in quantum barriers.
